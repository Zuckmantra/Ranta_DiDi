# Consumo de Gasolina DiDi - Gixxer 250

<p>
    Este proyecto es una calculadora de consumo de gasolina para el vehículo DiDi Gixxer 250.
    El usuario debe ingresar los kilómetros recorridos, el litros de gasolina consumidos y el precio del gasolina.
    La calculadora mostrará el consumo de gasolina en kilómetros por litro y el costo total de la gasolina.
</p>
